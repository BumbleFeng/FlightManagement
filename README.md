# Flight Management System

路	Developed a web application based on Spring MVC for airline, airport and passengers  
路	Mapped model to database using Hibernate and maintained data in MySQL  
路	Implemented user authentication and authorization in the use of Spring Security for password encryption, URL intercepting and CSRF protection  
路	Improved user experience by implementing auto-completion, multi-level checkbox and account name checking with Ajax and AngularJS
